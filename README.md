# Smp Mod(very creative I know)
The framework for your next smp, it's heavily inspired by craft attack 13 and aims to make your next two minecraft phase last longer.

# Features
### Disable Nether
A slow start helps to keep the server alive for longer

### Spawn Elytra
Makes it easier for players to visit each other and find a spot for their base

### No Sleep
Helps to build farms that require night time. For example the zombie in an iron farm

### No rocket boost in the overworld
Slows the game down a little

### Speedy ghast
Brings players closer together
